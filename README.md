# CommerceAutomation

This is a Selenium-based automation project for testing key user flows in a nopCommerce website.

## Features Automated

-  User Registration
-  Login / Logout
-  Checkout Process

## Tech Stack

- Java
- Selenium WebDriver
- TestNG / JUnit
- Maven
- IntelliJ IDEA

## How to Run the Tests

1. Clone the repo
2. Open in IntelliJ
3. Install dependencies (`mvn clean install`)
4. Run the test classes from `src/test/java`
